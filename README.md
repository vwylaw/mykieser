# Kieser Training Dashboard

A premium web application for tracking your Kieser training progress.

## Features

- **Dashboard**: Overview of all your assigned machines and recent training sessions.
- **Machine History**: Detailed graphs showing your weight and duration progress over time for each machine.
- **Session Analysis**: Inspect individual session performance.
- **Premium Design**: Dark mode interface designed for clarity and focus.

## Tech Stack

- React + TypeScript + Vite
- Recharts for data visualization
- Lucide React for iconography
- Vanilla CSS with CSS Variables for theming

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:5173` in your browser.

## Data

The application is currently seeded with sample data from the `kieser/sample` directory.
